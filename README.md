## Tech Stack:

### Android:
- Kotlin (Main Programming Language)
- Jetpack Compose (For UI Kit)
- Google Maps
- TFLite
- Firebase

### Backend:
- Golang (Main Programming Language)
- Gin (Framework)
- Heroku (Cloud Provider)
- Supabase (Database Cloud Provider)

### Machine Learning:
- Jupyter Notebook
- Python
- Tensorflow (With Object Detection Module)
